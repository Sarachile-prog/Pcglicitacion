# **App Name**: Licitaciones Globales

## Core Features:

- Web Scraping: Automatically scrape data from various government and institutional websites that post calls for bids.
- Categorization: Categorize bid opportunities based on pre-defined categories and industries.
- State information Extraction: Extract all State (government) information and entities related to each process or offering.
- Key Information Extraction: Extract and summarize key details from bid documents, including deadlines, amounts, and requirements, with reasoning tool to enhance comprehension.
- Dashboard Visualization: Visualize bid data and trends in a user-friendly dashboard.

## Style Guidelines:

- Primary color: Dark blue (#1E3A8A) to convey trust and professionalism.
- Background color: Light gray (#F5F5F5) to provide a clean, neutral backdrop.
- Accent color: Teal (#26A69A) to highlight important information and CTAs.
- Body and headline font: 'Inter', a sans-serif font, will be used throughout, providing a clean and modern feel.
- Use simple, professional icons to represent bid categories and data points.
- Maintain a clean and organized layout with clear information hierarchy for easy navigation.
- Implement subtle animations and transitions to enhance user experience and provide feedback during data loading.
